# Hospital Cost Analysis Project Overview 🏥

## Objective
Utilize Medicare cost report data obtained from the Department of Health and Human Services (HHS) to measure and interpret variations in inpatient hospital care cost across the United States.

## Main Focus Points
- Evaluate if there are correlations between costs and employee salaries
- Identify highest costs hospitals 
- Analyze varying hospital costs by state and rural vs. urban settings

## Data
- Original dataset contains 6,103 rows and 117 columns
